Основная цель:
Приложение создано для возможности учителем проставлять оценки студентам, а студентам просматривать данны учителя.

Приложение открывается с начальной страницы, где пользователь должен выбрать он заходит как студент или как преподаватель.

    1 вариант - вход как преподаватель (преподаватель может вносить данные и изменять значения в базе данных):
1.1 открывается новое окно авторизации для преподавателя
    валидные значения Логин: 1
                      Пароль: 1
1.2	Открывается страница с возможностью ввода данных имени, фамилии и номера группы студента
    пример заполнения Имя - John
                      Фамилия - Makori
					  Номер грппы  144-4
1.3 При нажатии кнопки "показать информацию" отобразится телефон и имеил студента
1.4 При выборе предмета ниже в текстовом поле отображается оценка и дата получения
    пример: physical educaton 
1.5 Преподаватель может поставить оценку в поле рядом с текстом "Поставить оценку", и затем нажать на кнопку Подтвердить.
    При нажатии кнопки Обновить оценки, новая оценка появится в ранее указанном текстовом поле.
1.6 Есть возможность изменить номер группы, 
    для этого в текстовом поле в нижнем правом углу надо ввести номер  новой группы	
	пример: 144-4, затем кнопка подтвердить
	новый номер группы автоматически появится в текстовом поле выше, и изменяться в базе данных.
	
	2 вариант- как студент:
2.1 Открывается окно с возможностью выбора предмета, при этом в поле автоматически отображаются данные о предмете.
2.2 Студент не имеет имеет возможность только просматривать данные базы данных.
